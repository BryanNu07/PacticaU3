# Aplicativo Interactivo de Lógica y Algoritmos en C++

## Descripción

Este proyecto fue desarrollado en el lenguaje de programación C++ como parte de la asignatura de Algoritmos y Lógica de Programación de la Universidad Técnica de Ambato.

La aplicación permite registrar estudiantes, ingresar sus calificaciones, calcular promedios automáticamente y determinar si cada estudiante aprueba o reprueba. Además, los resultados son almacenados en archivos de texto utilizando persistencia de datos.

---

## Funcionalidades

- Registro de hasta 40 estudiantes
- Ingreso de nombres y calificaciones
- Cálculo automático de promedios
- Identificación de aprobados y reprobados
- Uso de arreglos
- Manejo de archivos `.txt`
- Validación de datos
- Menú interactivo en consola
